## Installation

via [Alcatraz](https://github.com/supermarin/Alcatraz)

or manually install
- Copy the .dvtcolortheme to ``` ~/Library/Developer/Xcode/UserData/FontAndColorThemes/ ```.
- Restart Xcode.
- done :]
